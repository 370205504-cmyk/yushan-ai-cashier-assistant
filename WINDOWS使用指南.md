# 雨姗AI收银助手 v5.0.0 - Windows 使用指南

## 🚀 5分钟快速上手指南

### 第一步：解压软件
1. 将压缩包 `yushan-ai-cashier-assistant-v5.0.0-windows.zip 解压到电脑任意位置（推荐：D:\YushanAI）

### 第二步：启动服务
1. 双击 `启动服务.bat`
2. 等待约10秒，看到「服务已启动」提示
3. 浏览器会自动打开配置向导

### 第三步：配置系统
1. 按照向导5个步骤，5分钟内完成
2. 检测收银系统
3. 生成店铺码和桌码
4. 配置打印机
5. 完成！

## 🏪 配置向导详解

### 步骤1：检测本地收银系统
- 自动扫描您的电脑
- 自动识别已安装的收银系统
- 自动检测打印机
- 点击「下一步」继续

### 步骤2：配置收银系统对接
- 选择您的收银系统类型
- 建议选择「只读模式」（更安全）
- 输入数据库连接信息
- 点击「下一步」继续

### 步骤3：生成店铺码和桌码
- 输入店铺名称
- 设置桌位数量
- 自动生成所有二维码
- 一键打印二维码

### 步骤4：配置打印机
- 选择您的打印机型号
- 配置打印场景
- 设置打印份数
- 点击「下一步」继续

### 步骤5：完成配置
- 恭喜！系统已配置完成
- 打印店铺码和桌码
- 开始使用！

## 🛒 点餐系统使用

### 顾客扫码点餐
1. 顾客扫描桌上的二维码
2. 浏览菜单，选择菜品
3. 选择规格、口味、数量
4. 加入购物车
5. 确认下单

### 商家查看订单
1. 登录管理后台
2. 查看「待处理」订单
3. 点击「接单」「完成」
4. 订单自动同步到收银系统

### 订单状态说明
- 待确认：顾客刚下单
- 已接单：商家已确认
- 制作中：后厨正在制作
- 已出餐：菜品已完成
- 已取消：订单已取消

## 🔧 常见问题

### 问题1：启动失败怎么办？
- 检查是否安装了Node.js 18+
- 检查3000端口是否被占用
- 查看logs目录下的错误日志

### 问题2：收银系统连接失败？
- 检查数据库连接信息是否正确
- 确认数据库服务是否启动
- 尝试重启服务

### 问题3：打印机无法打印？
- 检查打印机是否连接
- 检查打印机驱动是否正常
- 尝试打印测试页

### 问题4：数据会上传云端吗？
- **不会！**所有数据本地加密存储
- 只在您的电脑上
- 不上传任何云端
- 安全可靠
